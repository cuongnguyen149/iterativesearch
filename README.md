#Must have:
###NodeJS and npm package: 
    install nodejs and npm package -> https://nodejs.org/en/download/
##Run project:
####1.Checkout or clone source code.
####2.Open terminal at project directory and run:
      npm install
####3.Then run:
     node index.js
